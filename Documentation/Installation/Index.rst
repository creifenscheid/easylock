============
Installation
============

Target group: **Developers, Integrators**

The extension can be installed via composer

.. code-block:: bash

   composer require itx/jobapplications

or via TYPO3-Extension-Repository.

Include Static TypoScript
-------------------------

#. Go to the root page of your installation

#. Open the sys_template entry

#. add the provided static template in tab "Includes" in field "Include static (from extensions)"
